Ans.

Window object & Document
  We have the following when JavaScript runs in a browser 
All the browser specific functions of JavaScript you would find an window object.
window object represents browser window and provides methods to control it. It is a global object
This window object is provided you by the browser not by the JavaScript

Document object models(DOM)
the document object give you access to the whole document
Dom represents the page content as HTML

   document.body   -> page body as object

   document.body.style.background ="green"  ->Change page background to green
